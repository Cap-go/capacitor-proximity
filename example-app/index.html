Capgo Proximity

Toggle native proximity monitoring and inspect the current device state.

This demo is meant for a real phone. On Android the current app window dims while the sensor is covered. On iOS the operating system handles the screen behavior once monitoring is enabled.

Tip: keep this app in the foreground while testing. Browsers and simulators usually do not expose a usable proximity sensor, so the demo is expected to report `available: false` there.

Output

Idle 
Tap "Refresh status" to inspect the current native state.
